📖 使用指南

快速了解 DevTools Station 的用法和技巧。

🔍 搜索工具

在首页顶部的搜索框中输入关键词即可快速找到工具。搜索支持：

  • 中文搜索: 输入「加密」找到所有加解密工具
  • 拼音搜索: 输入「md5」或「json」即可定位
  • 快捷键: 按 Ctrl+K (Mac: ⌘K) 快速聚焦搜索框

🛠️ 使用工具

所有工具的使用方式基本一致：

  1. 点击工具卡片进入工具页面
  2. 在输入框中输入文本或参数
  3. 选择需要的选项（如有）
  4. 点击 ⚡ 执行 按钮或按 Ctrl+Enter
  5. 查看结果，点击 📋 复制 复制到剪贴板
💡 提示: 大部分工具提供了示例输入，点击示例文本可自动填充到输入框。

📂 工具分类

  • 🔐 加密解密: MD5、SHA、AES、Bcrypt、HMAC 等
  • 📝 格式化工具: JSON、SQL、CSS、HTML、XML 格式化
  • 🔄 转换工具: 时间戳、进制、颜色、大小写、Unicode 转换
  • 🎲 生成器: UUID、密码、随机数、Lorem Ipsum、二维码
  • 📄 文本处理: 文本对比、正则测试、字数统计、去重、Base64
  • 🌐 网络工具: IP 查询、UA 解析、HTTP 状态码、URL 编解码
  • 💻 开发者工具: Cron 表达式、Git 命令、MIME 类型
  • 📟 编码解码: URL 编码、HTML 实体、摩尔斯电码

⌨️ 键盘快捷键

  • Ctrl+K / ⌘K - 聚焦搜索框
  • Ctrl+Enter / ⌘Enter - 执行当前工具
  • Esc - 关闭搜索下拉

🔌 API 调用

所有工具都提供 RESTful API，可通过 HTTP 请求直接调用。查看 /doc.html 获取完整 API 文档。

示例：

curl -X POST http://localhost:8088/api/tools/crypto/hash \
  -d "input=Hello World&algorithm=md5"
