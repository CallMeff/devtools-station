# DevTools Station - 部署文档

## 环境要求

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 17+ | 推荐 OpenJDK 17 LTS |
| Maven | 3.8+ | 构建工具 |
| MySQL | 8.0+ | 数据库 |
| 操作系统 | Windows/Linux/macOS | 跨平台 |

## 快速开始

### 1. 克隆项目
```bash
cd d:/workspace/devtools-station
```

### 2. 初始化数据库
```bash
# 使用 MySQL 命令行或 Navicat 等工具执行
mysql -u root -proot < docs/sql/init.sql
```

### 3. 修改配置（如需要）
编辑 `src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/devtools_station?...
    username: root
    password: root
```

### 4. 编译运行
```bash
# 编译
mvn clean package -DskipTests

# 运行
java -jar target/devtools-station-1.0.0.jar

# 或直接使用 Maven 插件
mvn spring-boot:run
```

### 5. 访问
- 首页: http://localhost:8088
- API文档 (Knife4j): http://localhost:8088/doc.html
- Swagger UI: http://localhost:8088/swagger-ui.html
