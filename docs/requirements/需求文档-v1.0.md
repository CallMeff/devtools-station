## 一、项目概述

### 1.1 项目背景
参考 it-tools / MikuTools 等优秀在线工具箱，结合 cursor.com 的暗黑专业设计风格，构建一款面向开发者的独立在线工具箱。

### 1.2 项目定位
- **目标用户**: 开发者、IT 从业人员、技术爱好者
- **核心价值**: 无需安装，打开即用，一站式解决开发中的常见工具需求
- **设计风格**: 暗黑专业风（参考 cursor.com），支持亮色/暗色双主题

### 1.3 技术选型
| 层级 | 技术 | 说明 |
|------|------|------|
| 后端框架 | Spring Boot 3.2 | Java 17 |
| ORM | MyBatis Plus 3.5 | 简化数据库操作 |
| 数据库 | MySQL 8.0 | 本地数据库 |
| 模板引擎 | Thymeleaf | 服务端渲染 |
| 工具库 | Hutool 5.8 | 通用工具 |
| API 文档 | Knife4j 4.4 | Swagger 增强 |
| 前端 | 原生 HTML/CSS/JS | 轻量无框架依赖 |

## 四、API 接口设计

### 4.1 工具分类
- `GET /api/categories` - 获取分类列表
- `GET /api/tools` - 获取工具列表
- `GET /api/tools/hot` - 获取热门工具
- `GET /api/tools/search?q=` - 搜索工具

### 4.2 加解密
- `POST /api/tools/crypto/hash` - 哈希计算
- `POST /api/tools/crypto/base64` - Base64编解码
- `POST /api/tools/crypto/urlcode` - URL编解码
- `POST /api/tools/crypto/aes` - AES加解密
- `POST /api/tools/crypto/bcrypt` - BCrypt哈希

### 4.3 格式化
- `POST /api/tools/format/json` - JSON格式化
- `POST /api/tools/format/sql` - SQL格式化
- `POST /api/tools/format/css` - CSS格式化

### 4.4 转换器
- `POST /api/tools/convert/timestamp` - 时间戳转换
- `POST /api/tools/convert/radix` - 进制转换
- `POST /api/tools/convert/color` - 颜色转换
- `POST /api/tools/convert/case` - 大小写转换

### 4.5 生成器
- `POST /api/tools/generate/uuid` - UUID生成
- `POST /api/tools/generate/password` - 密码生成
- `POST /api/tools/generate/random` - 随机数生成
- `POST /api/tools/generate/lorem` - Lorem文本生成

### 4.6 文本处理
- `POST /api/tools/text/diff` - 文本对比
- `POST /api/tools/text/regex` - 正则测试
- `POST /api/tools/text/count` - 字数统计
- `POST /api/tools/text/unique` - 文本去重

### 4.7 网络工具
- `POST /api/tools/network/ua` - UA解析
- `GET /api/tools/network/ip` - IP查询
- `GET /api/tools/network/httpstatus` - 状态码

### 4.8 开发工具
- `POST /api/tools/dev/cron` - Cron解析
- `GET /api/tools/dev/git` - Git速查
- `GET /api/tools/dev/mime` - MIME类型

## 五、部署说明

### 5.1 环境要求
- JDK 17+
- Maven 3.8+
- MySQL 8.0+

### 5.2 启动步骤
```bash
# 1. 初始化数据库
mysql -u root -proot < docs/sql/init.sql

# 2. 编译运行
mvn clean package -DskipTests
java -jar target/devtools-station-1.0.0.jar

# 3. 访问
# 首页: http://localhost:8088
# API文档: http://localhost:8088/doc.html
```
