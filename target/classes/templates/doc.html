📚 API 文档

DevTools Station 提供 RESTful API，可用于集成开发或自动化工作流

🔐 认证接口

POST
/api/auth/register

用户注册：{ "username": "...", "password": "...", "email": "..." }

POST
/api/auth/login

用户登录：{ "username": "...", "password": "..." }，返回 Token

GET
/api/auth/me

获取当前用户信息（需 Header: X-Auth-Token）

POST
/api/auth/change-password

修改密码：{ "oldPassword": "...", "newPassword": "..." }

POST
/api/auth/reset-password

重置密码：{ "email": "...", "verifyCode": "...", "newPassword": "..." }

🔐 加密解密

POST
/api/tools/crypto/hash

哈希计算：input=文本 & algorithm=md5|sha1|sha256|sha512

POST
/api/tools/crypto/aes

AES 加解密：input=文本 & key=密钥 & mode=encrypt|decrypt

POST
/api/tools/crypto/bcrypt

Bcrypt 操作：input=密码 & mode=hash|verify & hash=已存hash

POST
/api/tools/crypto/hmac

HMAC 签名：input=消息 & key=密钥 & algorithm=HmacSHA256|...

📝 格式化工具

POST
/api/tools/format/json

JSON 格式化：input=JSON文本 & mode=format|minify|validate

POST
/api/tools/format/sql

SQL 格式化：input=SQL语句 & mode=format|minify

POST
/api/tools/format/xml

XML 格式化：input=XML文本 & mode=format|minify

🔄 转换工具

POST
/api/tools/convert/timestamp

时间戳转换：input=时间戳或日期字符串 & mode=auto

POST
/api/tools/convert/excel2json

Excel转JSON：multipart/form-data, file=Excel文件 & mode=array|objects

💻 开发工具

POST
/api/tools/dev/jwt

JWT 解析：input=JWT Token 字符串（不验证签名）

POST
/api/tools/dev/cron

Cron 表达式：expression=Cron表达式字符串

📋 统一响应格式

{
  "code": 200,
  "message": "success",
  "data": { ... }
}

code=200 成功 | code=400 参数错误 | code=401 未登录 | code=500 服务器错误
